# Execução do frontend

- Certifique-se de ter o Node Package Manager (NPM) instalado;
- Execute o comando "npm install" na pasta "evento-br-frontend";
- Ao realizar um commit ou puxar modificações, é recomendável que esteja na pasta raíz do projeto como um todo;
- Caso ocorra erro de deploy, verifique se a root directory do projeto é a sub-pasta "evento-br-frontend".

# Documentação

Toda a documentação específica das partes do projeto está inclusa em "docs", na raíz do código.
